## 📝 Customization

### Colors
Edit the CSS variables in `style.css`:
```css
:root {
    --primary-color: #6366f1;      /* Indigo */
    --secondary-color: #ec4899;    /* Pink */
    --dark-bg: #0f172a;            /* Dark slate */
    /* ... other colors */
}
```

### Content
- Edit section text in `index.html`
- Update the navigation menu links
- Modify feature descriptions
- Add your contact information

### Download Links
Update the download button links in the "Download FazClient" section with actual download URLs.
